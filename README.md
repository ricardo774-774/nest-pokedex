# Ejecutar en desarrollo

1. Clonar repo
2. Ejecutar 
```
yarn install
```
3. Si no tienes Nest CLI instalado. (Si ya lo tienes, pasa directo al paso 4)
```
npm i -g @nestjs/cli
```
4. Levantar la base de datos
```
docker-compose up -d
```
